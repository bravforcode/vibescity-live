    Create `src/components/ui/VibeBanner.vue`. Pure CSS marquee — no JS animation, no `setInterval`.

    Props:
    - `text` (String, default: 'VIBE OF THE HOUR') — the scrolling text content
    - `visible` (Boolean, default: true)

    The marquee is achieved by duplicating the text span twice inside a flex container, then animating `transform: translateX` from 0 to -50%. This creates a seamless loop without JS.

    i18n: the `text` prop accepts a raw string. Callers in HomeView can pass `t('gamification.vibe_of_hour')` or the literal. Do NOT hardcode the text string inside the component — use the prop.

    Accessibility: `role="marquee"`, `aria-live="off"` (prevents screen reader spam).

    Full component:
    ```vue
    <script setup>
    defineProps({
      text:    { type: String,  default: 'VIBE OF THE HOUR' },
      visible: { type: Boolean, default: true },
    });
    </script>

    <template>
      <div
        v-show="visible"
        class="vibe-banner"
        role="marquee"
        aria-live="off"
        aria-atomic="false"
      >
        <div class="vibe-banner-track" aria-hidden="true">
          <span class="vibe-banner-item">{{ text }} &nbsp;✦&nbsp; </span>
          <span class="vibe-banner-item">{{ text }} &nbsp;✦&nbsp; </span>
        </div>
      </div>
    </template>

    <style scoped>
    .vibe-banner {
      position: absolute;
      top: 0;
      left: 0;
      right: 0;
      z-index: 500;
      overflow: hidden;
      height: 28px;
      background: rgba(0, 0, 0, 0.75);
      backdrop-filter: blur(4px);
      border-bottom: 1px solid rgba(0, 229, 255, 0.2);
      display: flex;
      align-items: center;
    }

    .vibe-banner-track {
      display: flex;
      white-space: nowrap;
      animation: vibe-marquee 18s linear infinite;
      will-change: transform;
    }

    .vibe-banner-item {
      font-size: 11px;
      font-weight: 900;
      letter-spacing: 0.12em;
      color: #00e5ff;
      text-shadow: 0 0 6px #00e5ff;
      text-transform: uppercase;
      padding-right: 40px;
    }

    @keyframes vibe-marquee {
      0%   { transform: translateX(0); }
      100% { transform: translateX(-50%); }
    }

    @media (prefers-reduced-motion: reduce) {
      .vibe-banner-track {
        animation: none;
      }
    }
    </style>
    ```

    Props:
