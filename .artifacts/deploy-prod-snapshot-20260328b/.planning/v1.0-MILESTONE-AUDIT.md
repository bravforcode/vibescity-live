## Critical Gap — Marker Tap → VibeActionSheet Broken

**Severity:** Critical (blocks core UX flow)

**Root cause:**
- `HomeView.vue` wires `@select-shop="handleMarkerClick"` (from MapboxContainer)
- `handleMarkerClick` in `useAppLogic.js` calls `applyShopSelection()` + `smoothFlyTo` but **never sets `selectedShop.value`**
- `VibeActionSheet` is gated on `:visible="!!selectedShop"` — stays hidden
- Only `handleOpenDetail` (triggered by `@open-detail`, not `@select-shop`) sets `selectedShop.value`

**Fix required:**
```
File: src/composables/useAppLogic.js
Function: handleMarkerClick (~line 405)
Add: selectedShop.value = shop  (after applyShopSelection call)
```
