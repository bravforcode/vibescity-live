  <files>src/components/ui/YouAreHere.vue</files>
  <action>
    Create `src/components/ui/YouAreHere.vue`. This is a purely cosmetic CSS overlay — no map API calls.

    Props:
    - `visible` (Boolean, default: true) — controls v-show
    - `label` (String, default: "YOU ARE HERE")

    Template: A fixed-position container (pointer-events: none) with:
    1. Outer ring: large semi-transparent blue circle that pulses opacity (CSS keyframes)
    2. Inner dot: solid blue circle (#00bfff), 14px, with a white center dot
    3. Label text below: "YOU ARE HERE" in cyan uppercase, 9px, monospace

    CSS rules:
    - Pulse animation uses `transform: scale()` and `opacity` ONLY — no box-shadow animation
    - `prefers-reduced-motion: reduce` disables animation

    Full component:
    ```vue
    <script setup>
    defineProps({
      visible: { type: Boolean, default: true },
      label: { type: String, default: 'YOU ARE HERE' },
    });
    </script>

    <template>
      <div
        v-show="visible"
        class="you-are-here"
        role="img"
        :aria-label="label"
      >
        <div class="yah-outer-ring" aria-hidden="true" />
        <div class="yah-dot" aria-hidden="true">
          <div class="yah-dot-inner" />
        </div>
        <span class="yah-label">{{ label }}</span>
      </div>
    </template>

    <style scoped>
    .you-are-here {
      position: absolute;
      display: flex;
      flex-direction: column;
      align-items: center;
      pointer-events: none;
      z-index: 200;
      transform: translate(-50%, -50%);
    }

    .yah-outer-ring {
      position: absolute;
      width: 48px;
      height: 48px;
      border-radius: 50%;
      border: 2px solid #00bfff;
      opacity: 0.5;
      animation: yah-pulse 2s ease-in-out infinite;
    }

    .yah-dot {
      width: 14px;
      height: 14px;
      border-radius: 50%;
      background: #00bfff;
      display: flex;
      align-items: center;
      justify-content: center;
      box-shadow: 0 0 8px #00bfff;
    }

    .yah-dot-inner {
      width: 5px;
      height: 5px;
      border-radius: 50%;
      background: #fff;
    }

    .yah-label {
      margin-top: 20px;
      font-size: 9px;
      font-weight: 900;
      letter-spacing: 0.12em;
      color: #00e5ff;
      text-shadow: 0 0 4px #00e5ff;
      font-family: monospace;
      white-space: nowrap;
    }

    @keyframes yah-pulse {
      0%, 100% { transform: scale(1); opacity: 0.5; }
      50%       { transform: scale(1.6); opacity: 0; }
    }

    @media (prefers-reduced-motion: reduce) {
      .yah-outer-ring {
        animation: none;
      }
    }
    </style>
    ```
