🔐 VibeCity Admin Quick Access

เครื่องมือช่วยเข้าหน้า Admin แบบง่ายๆ

⚠️ กรุณาตั้งค่า Environment ก่อนใช้

🚀 Quick Actions:

📋 Instructions:

  1. เลือก Environment (Local/Staging/Production)
  2. ใส่ Admin Email ที่ต้องการ
  3. Domain จะถูกตั้งค่าอัตโนมัติ
  4. กด "Generate Magic Link" เพื่อรับ Link สำหรับ Sign-in
  5. กด "Open Admin Page" เพื่อเปิดหน้า Admin โดยตรง

🔧 Manual Commands:

# Generate Magic Link
node scripts/generate-admin-magic-link.cjs admin@example.com DOMAIN/admin

# Grant Admin Access
node scripts/grant_admin.js admin@example.com

# Bootstrap Admin Users
node scripts/bootstrap-admin-users.cjs
